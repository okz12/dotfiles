# Linting

E vs. W:
- E: Errors
- W: Warnings

Status codes
- `100`: indentation
- `200`: whitespace
- `300`: blank lines
- `400`: imports
- `500`: line length
- `600`: deprecation
- `700`: statements
- `900`: syntax errors
